# 小红书一键创作助手

输入一个关键词，自动生成小红书完整内容：标题、正文、话题标签和精美配图。

---

## 快速开始

### 第一步：安装依赖

打开终端（命令提示符），输入：

```bash
pip install -r requirements.txt
```

### 第二步：启动程序

```bash
python app.py
```

### 第三步：打开浏览器

访问 **http://127.0.0.1:5000** 即可使用。

---

## 功能说明

| 功能 | 说明 |
|------|------|
| 文案生成 | 支持5种内容类型：教程攻略、好物测评、种草推荐、日常Vlog、清单合集 |
| 图片生成 | 自动生成封面图、内容卡片、标签页（暖色教育风） |
| 一键导出 | 所有内容保存到 `output/` 目录 |

## 常见问题

**Q: 图片生成失败？**
A: 确保系统安装了中文字体（微软雅黑），程序会自动检测。

**Q: 如何修改配色？**
A: 编辑 `image_generator.py` 中的 `COLORS` 字典。

**Q: 端口被占用？**
A: 修改 `app.py` 最后一行的 `port=5000` 为其他端口号。
